# 🧠 AgentFlow-AI

A step-by-step implementation of AI agents built from scratch — evolving from a rule-based system to a memory-enabled LLM agent.

This project focuses on understanding the **core architecture of AI agents** without relying on external frameworks like LangChain.

---

## 🚀 Features

✅ Rule-Based Agent  
✅ LLM-Only Agent  
✅ LLM + Tool-Using Agent  
✅ Conversational Memory Agent  
✅ Streamlit Chat Interface  

---

## 🏗️ Agent Evolution

### V1 - Rule-Based Agent
- Regex-based math detection
- Word counting tool
- Action → execution pipeline

### V2 - LLM-Only Agent
- Prompt-based reasoning
- Instruction-following responses

### V3 - LLM + Tools
- LLM selects tools
- Structured action parsing
- Dynamic tool execution

### V4 - Memory Agent
- Stores conversation history
- Context-aware responses
- Sliding window memory
